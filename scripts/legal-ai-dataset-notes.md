# 🧠 דאטא סט מוח משפטי AI — Factum-IL

דאטא סט זה מכיל את כל רכיבי המוח המשפטי של מערכת Factum-IL.
הקבצים מאורגנים לפי תחום ומתויגים בשמות עבריים ברורים.

> **המערכת הקיימת לא שונתה.** כל הקבצים כאן הם **עותקים בלבד** לצורך יצירת דאטא סט.

---

## רשימת הקבצים בריליס זה

| קובץ | תיאור |
|------|--------|
| `מנוע-AI-ישראלי.zip` | OllamaClient, שרשרת חשיבה 5 שלבים (הקשר → סיווג → סמכויות → סתירות → מסקנה), circuit breaker |
| `מסנני-בטיחות-AI.zip` | זיהוי PII, מניעת הזיות, בידוד הזרקות prompt, מגן חיסיון עו"ד–לקוח |
| `ניתוב-מודלים.zip` | בחירת מודל AI, בדיקת תקינות, מרשם מודלים |
| `אונטולוגיה-משפטית-ישראלית.zip` | היררכיית בתי המשפט הישראליים, טקסונומיה משפטית, נרמול מונחים, מילון נרדפים |
| `מנוע-אחזור-וחיפוש-וקטורי.zip` | חיפוש KNN (sqlite-vec), חיפוש היברידי FTS5+וקטור, חיתוך טקסט, הטמעות |
| `מנוע-אסמכתאות-ישראלי.zip` | פרסר ציטוטים ישראלי (Nevo 2021) — בג"ץ, ע"א, ת"פ, עב, תמש ועוד |
| `מערכת-הערכת-AI.zip` | harness הערכה, regression fixtures, מדדי דיוק וזכירה |
| `ליבת-סוכני-AI.zip` | ממשק בסיס סוכן, מרשם כלים, CaseExecutionContext, RBAC, יומן ביצוע |
| `מתאם-סוכני-AI.zip` | תזמור רב-סוכני, רצף שלבים, מנוע עקביות |
| `זיכרון-שיחה-לתיק.zip` | זיכרון שיחה לפי תיק (SQLite), assembler הקשר, הגנת פרטיות |
| `עיבוד-קורפוס-משפטי-וכנסת.zip` | עיבוד כנסת OData, WikiSource, case-law-israel (HuggingFace), הסקת קורפוס עליון |
| `ניתוח-סיכוני-ליטיגציה.zip` | ניקוד סיכוני מועדים, זיהוי סתירות, ניתוח פערי ראיות, גרף תלויות |
| `נתיבי-API-ומודולי-מוח-משפטי.zip` | כל נתיבי Express של המוח המשפטי + מודולים + שירותים + כלי עזר |
| `שאילתות-מסד-נתונים-משפטי.zip` | שאילתות TypeScript לכל טבלאות המוח המשפטי |
| `מיגרציות-סכמת-מוח-משפטי.zip` | קבצי SQL של מיגרציות בסיס הנתונים (069–085 + תשתית AI) |
| `סכמת-בסיס-הנתונים-המלאה.zip` | סכמת SQLite המלאה של המערכת |
| `README-דאטא-סט-מוח-משפטי.md` | מסמך זה |

## מבנה המוח המשפטי

```
packages/ai/                    ← מנוע AI ישראלי
packages/ai-guardrails/         ← מסנני בטיחות
packages/model-router/          ← ניתוב מודלים
packages/legal-ontology/        ← אונטולוגיה משפטית
packages/retrieval/             ← אחזור וחיפוש וקטורי
packages/citation-engine/       ← מנוע אסמכתאות
packages/evals/                 ← הערכת AI
packages/agent-core/            ← ליבת סוכנים
packages/orchestrator/          ← תזמור סוכנים
packages/memory/                ← זיכרון שיחה
packages/legal-corpus-ingest/   ← עיבוד קורפוס
packages/litigation-intelligence/ ← ניתוח ליטיגציה
```
