### תיקון 2: ATTACH path escaping
**קובץ:** `packages/database/src/connection.ts`
```diff
- this.db.exec(`ATTACH DATABASE '${dataPath}' AS data_store`);
+ this.db.exec(`ATTACH DATABASE '${dataPath.replace(/'/g, "''")}' AS data_store`);
```
מונע שבירת SQL כאשר ב-path יש apostrophe (למשל ב-Windows: `C:\Users\D'Angelo\...`).

### תיקון 3: שמות עמודות שגויים ב-recovery.ts
