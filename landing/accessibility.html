הצהרת נגישות

עדכון אחרון: יוני 2026

Factum-IL מחויבת לנגישות דיגיטלית ולהנגשת המוצר לאנשים עם מוגבלויות, בהתאם לחוק שוויון זכויות לאנשים עם מוגבלות, התשנ"ח–1998, ותקנות הנגישות לשירות.

1. מצב הנגישות

אנו עובדים לעמוד בדרישות תקן WCAG 2.1 ברמה AA. אתר הנחיתה של Factum-IL עומד בחלקו בדרישות אלה, ואנו ממשיכים לשפר את הנגישות באופן שוטף.

2. התאמות נגישות קיימות

  • כל הדפים מוגדרים עם lang="he" ו-dir="rtl" לתמיכה מלאה בעברית וכיווניות RTL
  • ניגודיות צבעים עומדת בדרישות WCAG AA (יחס של לפחות 4.5:1 לטקסט רגיל)
  • כל האלמנטים האינטראקטיביים נגישים למקלדת
  • תגיות סמנטיות HTML5 (header, main, footer, nav) בשימוש לאורך כל הדפים
  • גופנים בגודל קריא (בסיס 16px ומעלה)
  • ריווח מספק בין אלמנטים קליקביליים
  • אין תוכן מהבהב בתדירות שעלולה לגרום להתקפים אפילפטיים

3. אפליקציית שולחן העבודה

אפליקציית Factum-IL לשולחן העבודה (Electron/WebView2) מיועדת לעורכי דין ומשרדי עורכי דין. אנו עובדים על שיפורי נגישות לאפליקציה, כולל:

  • תמיכה בקוראי מסך (NVDA, JAWS) עבור ממשק המשתמש הראשי
  • ניווט מלא באמצעות מקלדת בכל פעולות הליבה
  • הגדלת טקסט ללא שבירת פריסה (עד 200%)
  • תמיכה במצב ניגודיות גבוהה של Windows

4. חסמי נגישות ידועים

להלן חסמי נגישות ידועים בהם אנו עובדים לטפל:

  • חלק מהגרפים והוויזואליזציות בלוח הבקרה אינם מנגישים עדיין לקוראי מסך — בטיפול
  • ממשק העלאת קבצים עדיין לא מוצהר במלואו ל-ARIA — בטיפול
  • כמה הודעות שגיאה אינן מכריזות עצמן אוטומטית לקורא מסך — בטיפול

5. טכנולוגיות נגישות נתמכות

המוצר נבדק ומיועד לעבוד עם:

  • דפדפנים: Chrome, Edge (גרסאות עדכניות)
  • קוראי מסך: NVDA עם Firefox, JAWS עם Chrome
  • הגדלת מסך: Windows Magnifier, ZoomText
  • ניווט מקלדת: Tab, Shift+Tab, Enter, Space, Arrow keys

6. פנייה בנושא נגישות

אם נתקלתם בבעיית נגישות או שהמוצר אינו נגיש לכם, אנא פנו אלינו. אנו מתחייבים להגיב תוך 5 ימי עסקים ולטפל בבעיה בהקדם האפשרי.
